Publishing Online

Right now, all your files (aka your website) is stored locally onto your computer. In order to upload it to the internet so the whole world can see, we have to deploy our website online!

Downloading Git

  • Git is the website we’re going to use to publish our website

  • (Download Git Here)[https://git-scm.com/download/win]

Creating your Github Account and Online Repository

  • Go to Github and create a new account

  • Your username will be the URL of your site – I recommend using your name

  • Click the plus sign in the upper right and click create new repository

  • Name your new repository username.github.io, except replacing username with your own username (eg: mine would be gracejiang.github.io)

Creating your Local Repository

  • Open terminal (Mac) or command line (Windows)

  • Copy and paste git clone https://github.com/username/username.github.io, replacing the username with your own username

    • Eg: mine would be git clone https://github.com/gracejiang/gracejiang.github.io

  • Enter your project folder using cd username.github.io

    • Eg: mine would be cd gracejiang.github.io

  • Create the home page of your project using echo "Hello World!!!" > index.html

Pushing your Local Changes

  • Copy and paste each line into your terminal / command line

  • git add --all

  • git commit -m "Initial commit"

  • git push -u origin master

Checking Online

  • Go to https://username.github.io (replacing username with your own username)

  • Eg: mine is https://gracejiang.github.io

Uploading Your Website

  • Now go into the your username.github.io folder

  • Delete the index.html file from it

  • Copy and paste in all your files from your original folder

  • Again, copy and paste each of the following lines into your terminal / command line:

    • git add --all

    • git commit -m "Uploading Website"

    • git push -u origin master

Your New Site

  • Changes to live will usually take a while, so check back to your website in 5-10 minutes

  • Your website is now live at https://username.github.io (replacing username with your own username)

Congrats!

  • Congratulations – you officially have your own personal website up and running on the internet! Thanks for sticking along :)
